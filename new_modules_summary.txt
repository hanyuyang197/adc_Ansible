## 第一批新增模块和 Action (资源统计相关)

### 1. adc_slb_resource_statistics.py
- slb_resource_statistics_list - 获取SLB资源统计信息

- profile_resource_statistics_list - 获取Profile资源统计信息

- ssl_resource_statistics_list - 获取SSL资源统计信息

## 第二批新增模块和 Action (系统管理相关)

### 5. adc_system_operations.py (扩展现有模块)
新增以下 action:
- admin_disable_get - 获取管理员状态 (API: admin.disable.get)
- admin_disable_edit - 禁用管理员用户 (API: admin.disable.edit)
- admin_two_factor_authentication_get - 获取双因素认证状态 (API: admin.two_factor_authenication.get)
- admin_two_factor_authentication_set - 开启双因素认证 (API: admin.two_factor_authenication.set)

原有的 action:
- save - 保存配置
- operation_reboot - 系统重启 (API: system.action.reboot)
- operation_reload - 重新加载配置 (API: system.action.reload)
- operation_shutdown - 系统关机 (API: system.action.shutdown)

### 6. adc_system_image.py (全新模块)
- system_image_list - 获取版本镜像列表 (API: system.image.list)
- system_image_apply - 指定镜像 (API: system.image.apply)
- system_image_upload - 通过 FTP/TFTP 方式上传镜像 (API: system.image.upload)

## 第三批新增 Action (SLB虚拟地址相关)

### 7. adc_slb_va.py (扩展现有模块)
新增以下 action:
- slb_va_stat_list - 获取虚拟地址状态列表 (API: slb.va.stat.list)
- slb_va_stat_get - 获取虚拟地址状态详情 (API: slb.va.stat.get)
- slb_va_stat_clear - 清除虚拟地址状态统计 (API: slb.va.stat.clear)

原有的 action:

- slb_va_get - 获取虚拟地址详情
- slb_va_add - 添加虚拟地址
- slb_va_edit - 编辑虚拟地址
- slb_va_del - 删除虚拟地址

## 详细参数说明

### admin.disable.edit 参数
- enable (int, 必选): 0=使用admin用户, 1=禁用admin用户
- name (str, 必选): 用户名
- password (str, 可选): 密码，缺省值: 不修改

### admin.two_factor_authenication.set 参数
- tfaenable (int, 必选): 0=不开启双因素认证, 1=开启双因素认证
- admintfaenable (int, 必选): 0=不开启双因素认证, 1=开启双因素认证

### system.image.apply 参数
- file_name (str, 必选): 指定下次启动使用的镜像文件的名称

### system.image.upload 参数 (FTP/TFTP)
- host (str, 必选): FTP/TFTP 服务器的 IPv4 地址
- location (str, 必选): 镜像名称，镜像必须处于 FTP/TFTP 的根目录
- protocol (str, 必选): 使用的协议，仅支持 "FTP" 和 "TFTP"
- port (int, 可选): 使用的端口号，FTP 缺省 21, TFTP 缺省 69
- username (str, 可选): 用户名，缺省值: 空字符串
- password (str, 可选): 密码，缺省值: 空字符串
- use_mgmt_port (int, 可选): 使用管理口，缺省值: 0
- reboot (int, 可选): 上传完镜像后是否自动重启，缺省值: 1
- save_config (int, 可选): 自动重启时是否保存配置，缺省值: 0
